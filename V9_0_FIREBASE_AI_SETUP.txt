WORDPILOT V9 — FIREBASE FUNCTIONS + SECRET MANAGER KURULUMU

ÖNEMLİ
- OpenAI anahtarını GitHub'a, index.html'e, security-config.js'e, ekran görüntüsüne
  veya sohbet mesajına yazma.
- Anahtar yalnızca Firebase Secret Manager komutuna yapıştırılmalıdır.
- Cloud Functions ve dış API çağrıları için Firebase projesinde faturalandırma
  hesabı/Blaze planı gerekebilir. Bütçe uyarısı kurmadan dağıtım yapma.

1. Bilgisayarda proje klasörünü aç.
2. Firebase CLI oturumu:
     firebase login
     firebase use wordpilot-7a574
3. Functions bağımlılıkları:
     cd functions
     npm install
     cd ..
4. Gizli anahtarı Secret Manager'a kaydet:
     firebase functions:secrets:set OPENAI_API_KEY
   Komut anahtarı istediğinde terminale yapıştır. Anahtar dosyaya yazılmaz.
5. Yalnız Functions dağıt:
     firebase deploy --only functions
6. Ardından Hosting dağıt:
     firebase deploy --only hosting
7. WordPilot'a giriş yap. Coach ekranındaki durum otomatik olarak
   “Güvenli Bulut AI hazır” olmalıdır.
