GitHub yükleme:
1. ZIP dosyasını aç.
2. İçindeki tüm dosyaları mevcut repository ana dizinine yükle.
3. Aynı isimli dosyaların üzerine yazılmasına izin ver.
4. Commit changes yap.
5. 1–3 dakika bekle.
6. Safari/Chrome sayfasını yenile.
