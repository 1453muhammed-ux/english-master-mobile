ENGLISH MASTER MOBILE TEST v1

Bu paket iPhone/iPad ana ekranına kurulabilen PWA test sürümüdür.

İçerik:
- 6000 kelime verisi Excel dosyasından alındı
- Arama ve grup filtresi
- Kelime ayrıntıları
- Cihazın İngilizce ses motoruyla dinleme
- Öğren / Ezber / Zor durumları
- Doğru bilinen kelimeyi aynı oturumda tekrar sormayan akıllı quiz
- Yanlış kelimeyi sonraki soruların ardından yeniden getirme
- Çevrimdışı çalışma

ÖNEMLİ: iPhone/iPad kurulumu için klasörün HTTPS bir web adresinde yayınlanması gerekir. GitHub Pages, Netlify veya kendi alan adınız kullanılabilir. Dosyayı doğrudan telefona kopyalamak PWA kurulumu sağlamaz.
