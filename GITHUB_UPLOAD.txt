WORDPILOT v4.3.1 GITHUB YÜKLEME

1. WordPilot_v4.3.1_GitHub_Ready.zip dosyasını bilgisayarınızda çıkartın.
2. GitHub deposunu açın: https://github.com/1453muhammed-ux/english-master-mobile
3. Add file > Upload files seçin.
4. Çıkartılan klasörün İÇİNDEKİ tüm dosyaları ve modules klasörünü yükleme alanına sürükleyin.
5. .github/workflows/firebase-hosting-live.yml dosyasını silmeyin. Pakette bulunan .github klasörü görünmese bile depodaki mevcut workflow korunmalıdır.
6. Commit changes düğmesine basın.
7. Actions > Deploy WordPilot to Firebase Hosting işleminin yeşil tik almasını bekleyin.
8. Şu bağlantıyı açın: https://wordpilot-7a574.web.app/?wpbuild=4.3.1
9. Eski sürüm görünürse bir kez https://wordpilot-7a574.web.app/reset-cache.html açın ve ardından 4.3.1 bağlantısına dönün.

ÖNEMLİ: modules/v431.js mutlaka GitHub'a yüklenmelidir.
