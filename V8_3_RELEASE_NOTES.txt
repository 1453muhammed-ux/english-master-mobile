WORDPILOT v8.3.0 — CONVERSATION COACH 4.0 TESTER BETA

1. Conversation Coach 4.0
- Introduced Mira as the conversation-coach persona.
- Added ten structured real-life role-play scenarios:
  introduction, airport, hotel, restaurant, shopping, doctor,
  university, job interview, directions and emergency assistance.
- Each scenario now has a location, role, goal, mission checklist,
  useful phrases and a defined number of conversation steps.
- Scenario progress is stored separately for each active course.
- Added a session-completion report with grammar, clarity, fluency,
  average score, answer count and completed missions.

2. More stable scoring
- Conversation scores are smoothed between turns so a single answer does
  not create an extreme jump or drop.
- Spoken answers are not penalised for missing capital letters or
  punctuation created by browser speech recognition.
- Existing gentle, fluency and teacher correction modes remain available.

3. Voice experience
- Coach replies prefer a suitable female system voice when the device
  exposes one. The exact voice still depends on the operating system and
  browser.
- Existing speed and thinking-pause controls remain active.
- The browser microphone can be used for spoken replies.

4. Pronunciation and basic voice analysis
- Added Word, Sentence and Scenario target modes.
- Added transcript-based metrics for pronunciation accuracy,
  completion, fluency and recognition confidence.
- Added word-level target highlighting and focused retry advice.
- Pronunciation history now stores the extended metrics.
- Audio recordings are not stored by WordPilot.
- This is not phoneme-level or clinical acoustic analysis. It evaluates
  the browser transcript, duration and recognition confidence.

5. Cloud AI readiness
- Updated the protected Firebase function prompt to Conversation Coach 4.0.
- Cloud AI remains disabled in security-config.js until the owner deploys
  Functions, stores OPENAI_API_KEY in Secret Manager and deliberately
  enables aiEnabled.
- No private API key is included in the browser package.

6. Preserved systems
- 2000-word English bank, CEFR data and semantic relations remain.
- Academy and Reader remain enabled.
- Existing user progress, course PP and Firebase/App Check settings remain.
