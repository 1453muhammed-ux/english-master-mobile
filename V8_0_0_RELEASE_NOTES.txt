WORDPILOT v8.0.0 — RELEASE NOTES

CONVERSATION COACH 3.0
- Patient microphone flow: 5-second default thinking pause, selectable 3/5/8 seconds.
- Recognition restarts after an unexpected browser stop while the session remains active.
- Manual second tap ends recording immediately.
- Coach voice defaults to a slower 0.72x rate.
- Voice answers are not scored down for automatic punctuation/capitalisation omissions.
- Correction cards now separate grammar, clarity and fluency.
- More specific Turkish and English local correction rules were added.
- Conversation history is used to choose a new contextual follow-up and avoid repeated questions.
- A visible thinking state appears before the coach answer.

LANGUAGE ROUTE
- Added a one-tap ⇄ native/target language swap.
- Added a top-right × close button.
- Applying a route no longer reloads the page.
- The selected target immediately becomes the active home course and the matching league language.

LEAGUE MIGRATION
- Every cloud row now carries coursePoints per language.
- League tabs read only the selected language's XP.
- Legacy total XP is no longer shown as if it belonged to English, Russian, Uzbek or another individual course.
- An unsynchronised legacy row displays 0 and a migration note rather than a false score.
- Each tester account should open V8 once while signed in so its course-specific values are written to Firestore.

PRESERVED
- Firebase Authentication and sync
- App Check integration
- Feedback & Ideas collection
- Existing progress key: wordpilot_v34
- PWA and offline cache
- 1000 clean concepts, 1500 Russian entries and 1000 Uzbek entries

STATUS
Tester Beta. Cloud AI remains disabled in the frontend until its backend is separately configured and deployed.
