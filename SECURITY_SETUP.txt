WORDPILOT v9.9.2 — FIREBASE SECURITY SETUP
==========================================

1. Firebase Authentication hesabının UID değerini alın.
2. Firestore'da admins koleksiyonunu açın.
3. Belge kimliği olarak aynı UID değerini kullanın.
4. Şu alanları ekleyin:

role        string   owner
active      boolean  true
displayName string   Muhammed Aslan

ROL YETKİLERİ
- owner: tam yetki ve feedback silme
- admin: feedback ve içerik yönetimi
- support: feedback yönetimi
- editor: feedback okuma, ileride içerik yönetimi
- viewer: salt okunur yönetim görünümü

GÜVENLİK KURALLARI
Bu paketteki firestore.rules dosyası Hosting ile birlikte yayımlanmalıdır:

firebase deploy --only firestore:rules,hosting

Tamamlandı veya Kapatıldı durumundaki feedback kayıtlarında publicReply alanı zorunludur.
Her yönetim güncellemesi feedback/{id}/history alt koleksiyonuna değiştirilemeyen bir kayıt ekler.
